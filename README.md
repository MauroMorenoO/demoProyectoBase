## Instalación
Instalar dependencias basicar para la correcta instalacion del proyecto.
#### Para Ambiente Local
```sh
cd dillinger
npm i
node app
```

#### Para Ambiente de Desarrollo
```sh
cd dillinger
npm i
node app
```

#### Para Ambiente de Producción

```sh
npm install --production
NODE_ENV=production node app
```
